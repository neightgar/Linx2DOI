# 🚀 Roadmap - План развития Linx2DOI

## Планируемые функции

### 📌 Высокий приоритет

#### 1. Функционал для выбранных элементов (колонка "Выбрать")

**Минимальный набор:**
- **"📥 Экспорт выбранных в RIS"** - экспорт только отмеченных записей в Mendeley
  - Применение: Выборочный экспорт из большого списка (например, 25 из 100 статей)
  - Сложность: Низкая
  - Польза: ⭐⭐⭐⭐⭐

- **"📋 Копировать DOI выбранных"** - скопировать все DOI в буфер обмена
  - Применение: Быстрое получение списка DOI для вставки в другие программы
  - Сложность: Очень низкая
  - Польза: ⭐⭐⭐⭐

- **"🔄 Обновить выбранные DOI"** - повторный поиск DOI только для выбранных
  - Применение: Исправление ошибочно определенных DOI без полной переобработки
  - Сложность: Средняя
  - Польза: ⭐⭐⭐⭐

**Расширенный набор (опционально):**
- **"📄 HTML из выбранных"** - создать отдельный HTML-отчет только с выбранными записями
- **"🗑️ Удалить выбранные"** - убрать отмеченные строки из таблицы
- **"✨ Очистить ручные DOI"** - сбросить ручные DOI у выбранных элементов

## 🐛 Известные ограничения

1. Парсинг работает только с нумерованными списками
2. Не все источники имеют DOI
3. CrossRef API может не находить некоторые статьи
4. Требуется подключение к интернету
