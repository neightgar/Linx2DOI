# 📝 Changelog

Все значимые изменения в проекте Linx2DOI документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.5.0] - 2026-02-07

### ✨ Добавлено

- **Экспорт в RIS формат**
  - Новая кнопка "📥 Экспорт в RIS"
  - Поддержка Mendeley, Zotero, EndNote, Papers
  - Автоматический парсинг метаданных из APA цитат
  - Экспорт авторов, заголовков, журналов, DOI, года, страниц

- **Обнаружение дублей DOI**
  - Автоматическая проверка повторяющихся DOI
  - Статус "ДУБЛЬ № X" с указанием первого вхождения
  - Оранжевый цвет для дублей в таблице
  - Отображение дублей в HTML с желтым фоном
  - Статистика дублей в HTML отчете

- **Переключение тем**
  - Светлая тема (по умолчанию)
  - Темная тема
  - Кнопка переключения в шапке
  - Автоматическое сохранение выбора
  - Полностью адаптированный интерфейс

- **Делегат для ручного DOI**
  - Выравнивание текста вверху ячейки при редактировании
  - Улучшенный UX для ввода длинных DOI

### 🎨 Изменено

- **Интерфейс**
  - Убран заголовок "Управление" над панелью управления
  - Убран заголовок "Результаты обработки" над таблицей
  - Более чистый и компактный вид
  - Иконка приложения в шапке (слева от названия)
  - Убран градиент в шапке (темный однотонный фон)
  - Увеличена высота строк таблицы до 45px
  - Увеличен прогрессбар: 300×22px (видны проценты)
  - Увеличена колонка "Ручной DOI" до 250px
  - Уменьшена минимальная ширина секций таблицы до 30px
  - Уменьшена колонка "Статус" до 80px

- **HTML результат**
  - Заголовки статей теперь кликабельны (ссылка на DOI)
  - Убрана отдельная строка с URL статьи
  - Добавлены стили hover для заголовков-ссылок
  - Добавлены оранжевые badges для дублей

- **Таблица результатов**
  - Скрыта колонка "Выбрать" (зарезервирована для будущего)
  - Все колонки растягиваются на ширину окна
  - Статус "ДУБЛЬ" отображается оранжевым цветом (#ce9178)

### 🐛 Исправлено

- Исправлена ошибка с кнопкой "Выбрать файл" (не было connect)
- Исправлены дублирующиеся вызовы setEnabled для кнопок
- Исправлена ошибка с использованием export_ris_btn до создания

### 🔧 Техническое

- Новый модуль `src/core/ris_exporter.py`
- Новый класс `ManualDOIDelegate` в `delegates.py`
- Новый класс `LightColors` для светлой темы
- Метод `toggle_theme()` для переключения тем
- Улучшена архитектура с использованием self.colors
- Добавлены проверки на дубли в `html_generator.py`

## [1.0.0] - 2025-XX-XX

### ✨ Добавлено

- **Основной функционал**
  - Парсинг .docx документов
  - Автоматический поиск DOI через:
    - Прямые DOI ссылки
    - PubMed (PMID)
    - PMC (PMCID)
    - ResearchGate
    - CrossRef поиск по заголовку
  - Генерация APA цитат (7-е издание)
  - HTML отчеты с библиографией
  - Ручной ввод DOI

- **Интерфейс**
  - PyQt6 графический интерфейс
  - Темная тема
  - Таблица результатов
  - Прогресс-бар обработки
  - Вкладки "Результаты" и "Лог"
  - Кнопки управления:
    - Выбор файла
    - Запуск обработки
    - Применить ручные DOI
    - Открыть результат
    - Справка

- **Модульная архитектура**
  - `src/core/` - бизнес-логика
  - `src/gui/` - интерфейс
  - Разделение ответственности
  - Переиспользуемые модули

### 🔧 Техническое

- Python 3.8+ совместимость
- Зависимости:
  - python-docx (парсинг .docx)
  - requests (HTTP запросы)
  - PyQt6 (GUI)
- QSettings для сохранения настроек
- Worker thread для фоновой обработки
- Signal-slot архитектура
- Rate limiting для API
